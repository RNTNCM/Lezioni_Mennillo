# Settare ambiente Linux per Git
1. Installare git sulla macchina con `sudo apt install git`
2. Settare username e password di github con i seguenti comandi
`git config --global user.name "Username Github"`
`git config --global user.email "Email Github"`

# Comandi Git

- `git commit` questo comando prepara i file in una sesione di staging
- `git push` questo comando serve ad inviare il codice in remoto
- `git pull` questo comando serve a recuperare i file da remoto
- `git marge` questo comando serve a fondere i branches di un progetto
